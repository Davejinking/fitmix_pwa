## 🟠 High (핵심 기능 오작동)

### BUG-032: 운동 카드 일괄 축소 기능 없음
- **화면:** 캘린더 → 운동 계획 페이지
- **재현:** 여러 운동을 추가하고 각 운동 카드가 확장된 상태
- **예상:** 모든 운동 카드를 한 번에 축소할 수 있는 버튼/기능
- **실제:** 각 운동 카드를 개별적으로만 축소 가능
- **심각도:** 🟡 Medium
- **상태:** 🆕 신규
- **카테고리:** 🎨 UI
- **문제점:**
  - 운동이 많을 때 스크롤이 길어짐
  - 전체 운동 목록을 한눈에 보기 어려움
  - UX 불편함
- **개선안:**
  - AppBar에 "모두 축소" / "모두 확장" 버튼 추가
  - 또는 운동 카드 헤더에 토글 버튼 추가
  - 상태 저장 (다음 방문 시 유지)

### BUG-034: 캘린더에서 "운동 시작" 클릭 시 불필요한 운동 계획 화면 거쳐감
- **화면:** 캘린더 페이지 → 운동 계획 페이지 → 운동 페이지
- **재현:** 
  1. 캘린더 페이지에서 "ワークアウト開始" 버튼 클릭
  2. 운동 계획 페이지로 이동
  3. 다시 "ワークアウト開始" 버튼 클릭해야 실제 운동 시작
- **예상:** 캘린더에서 바로 운동 페이지로 이동하여 즉시 운동 시작
- **실제:** 중간에 운동 계획 페이지를 거쳐야 함
- **심각도:** 🟠 High
- **상태:** 🆕 신규
- **카테고리:** 🔧 기능
- **문제점:**
  - 불필요한 화면 이동
  - 사용자가 두 번 버튼을 눌러야 함
  - UX 낭비 (클릭 2회 → 1회로 줄여야 함)
  - 운동 계획 페이지는 편집 목적이지 시작 목적이 아님
- **개선안:**
  - 캘린더의 "운동 시작" 버튼 클릭 시 직접 `WorkoutPage`로 이동
  - 운동 계획 페이지는 편집 목적으로만 사용
  - 또는 캘린더에서 운동 계획 페이지로 이동 시 자동으로 운동 시작

### BUG-033: 운동 카드 세트 추가 시 카드 높이 자동 조정 안 됨
- **화면:** 캘린더 → 운동 계획 페이지

- **예상:** 새로운 세트가 보이도록 자동 스크롤 또는 카드 높이 조정
- **실제:** 새로운 세트가 화면 아래로 숨겨짐 (가시화 안 됨)
- **심각도:** 🟡 Medium
- **상태:** 🆕 신규
- **카테고리:** 🎨 UI
- **문제점:**
  - 사용자가 추가한 세트를 바로 볼 수 없음
  - 세트 입력 시 스크롤 필요
  - UX 저하
- **개선안:**
  - 세트 추가 시 해당 위치로 자동 스크롤
  - 또는 ListView의 `animateTo()` 사용

### BUG-030: "최근 기록" 버튼 미구현
- **화면:** 캘린더 → 운동 계획 → 운동 세트 카드
- **재현:** 운동 세트 카드에서 "최근 기록" 버튼 탭
- **예상:** 해당 운동의 이전 기록(무게, 횟수) 표시

- **심각도:** 🟡 Medium
- **상태:** 🆕 신규
- **카테고리:** 🔧 기능
- **문제점:**

  - SessionRepo에서 해당 운동의 최근 기록 조회

### BUG-029: 운동 삭제 시 세트만 삭제되고 운동은 남아있음
- **화면:** 캘린더 → 운동 계획 페이지 (plan_page.dart)
- **재현:** 
  1. 운동 추가 후 세트 입력
  2. 세트 행의 빨간 삭제 버튼(⊖) 클릭
  3. 모든 세트를 삭제해도 운동 카드는 그대로 남아있음
- **예상:** 마지막 세트 삭제 시 운동 자체도 함께 삭제되거나, 별도의 운동 삭제 버튼 제공
- **실제:** 세트 입력란만 사라지고 빈 운동 카드가 남아있음 (총 볼륨 0kg 상태)
- **심각도:** 🟠 High
- **상태:** 🆕 신규
- **카테고리:** 🔧 기능
- **문제점:**
  - 사용자가 운동을 완전히 제거할 방법이 없음
  - 빈 운동 카드가 계속 남아서 UI 혼란 야기
  - 운동 계획에서 불필요한 항목이 누적됨
- **개선안:**
  - 마지막 세트 삭제 시 운동 전체 삭제 확인 다이얼로그 표시
  - 또는 운동 카드 헤더에 별도의 운동 삭제 버튼 추가
  - 빈 운동 카드 자동 정리 기능

   - "This Week Performance" / "今週のパフォーマンス"
   - "Total Records" / "総記録"

**예상 결과:** 모든 텍스트가 선택한 언어로 정확히 표시됨

### BUG-009: 업그레이드 페이지 접근 테스트

**테스트 단계:**
1. 홈 화면에서 우상단 별(⭐) 아이콘 확인
2. 별 아이콘 탭
3. 업그레이드 페이지로 이동되는지 확인
4. 업그레이드 페이지에서 "프리미엄으로 업그레이드" 제목 확인
5. "파워 상점" 버튼 탭하여 파워 상점으로 이동 가능한지 확인

**예상 결과:** 홈에서 업그레이드 페이지로 원활하게 접근 가능

### BUG-010: UserInfoFormPage 생년월일 UI 테스트

**테스트 단계:**
1. 홈 → 프로필 아이콘 → 프로필 페이지 이동
2. "수정" 버튼 탭하여 UserInfoFormPage 진입
3. "생년월일" 필드 탭
4. 날짜 선택 모달이 정상적으로 표시되는지 확인
5. 날짜 선택 후 "확인" 버튼 탭
6. 선택한 날짜가 필드에 정상 표시되는지 확인

**예상 결과:** UI 깨짐 없이 정상적인 날짜 선택 인터페이스 표시

### BUG-011: 운동 세트 카드 i18n 테스트

**테스트 단계:**
1. 시뮬레이터 언어를 English 또는 日本語로 변경
2. 캘린더 → 운동 계획하기 → 운동 추가
3. 운동 선택 후 세트 카드에서 다음 텍스트 확인:
   - "Total Volume 0kg" / "総ボリューム 0kg"
   - "Recent Record" / "最近の記録"
4. 세트 추가 후 무게/횟수 입력
5. 총 볼륨이 계산되어 해당 언어로 표시되는지 확인

**예상 결과:** 세트 카드의 모든 텍스트가 선택한 언어로 표시됨
