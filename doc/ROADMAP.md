## 📅 최근 업데이트 (2026-01-21)

### ✅ 루틴 태그 색상 시스템 개선
- 사용자 선택 색상 저장 기능 추가 (Routine 모델에 tagColors 필드)
- 3단계 색상 우선순위 구현: 시스템 프리셋 > 사용자 색상 > 해시 기반 자동 생성
- 커스텀 태그 생성 후 UI 즉시 새로고침 문제 해결
- 사용자가 선택한 색상이 필터 바에 정확히 표시되도록 수정

### ✅ 홈 화면 성능 위젯 추가
- **Weekly Status**: 클릭 가능하게 변경 (Analysis 화면으로 이동)
- **Big Three Widget**: Bench/Squat/Deadlift 최대 중량 합계 표시
- **Volume Widget**: 주간 총 볼륨 표시 (kg 또는 ton)
- 전술적 HUD 스타일 적용: 투명 배경, 날카로운 모서리, 얇은 테두리
- 네온 색상 사용 (Big 3: 그린, Volume: 블루)

### ✅ 홈 화면 레이아웃 최적화
- Law of Proximity 적용: Weekly Status와 Session Info 간격 대폭 축소 (24px → 16px)
- Top-heavy 레이아웃: 주요 액션을 화면 상단에 집중
- 불필요한 여백 제거 (내부 60px 간격 삭제)
- 일관된 24px 간격으로 섹션 구분
- 응집력 있는 대시보드 느낌 구현

## 📋 프로젝트 개요

**Iron Log**는 개인 운동 기록에 특화된 심플하고 강력한 웨이트 트레이닝 앱입니다.
- **Iron (쇠)**: 무게, 강철 같은 의지
- **Log (기록)**: 운동 일지, 데이터 로그

---

## 🎯 개발 단계 (Development Phases)

### 📍 Phase 1: Athlete 모드 & MVP (현재 단계) ⭐
> **목표:** 혼자 운동하는 사용자를 위한 완벽한 기록 앱

**핵심 기능:**
- [x] 운동 계획 및 기록
- [x] 휴식 타이머 (애플 스타일)
- [x] 템포 모드
- [x] 게이미피케이션 (XP, 레벨, 스트릭)
- [ ] Iron Pro 구독 기능
- [ ] 고급 통계 차트
- [ ] 데이터 백업/복원

### 📍 Phase 2: Squad 모드 (확장)
> **목표:** 커뮤니티 및 경쟁 기능

**핵심 기능:**
- [ ] 친구 추가 및 팔로우
- [ ] 운동 기록 공유
- [ ] 주간/월간 리더보드
- [ ] 챌린지 참여

### 📍 Phase 3: Coach 모드 (수익화 확장)
> **목표:** 트레이너를 위한 SaaS 플랫폼

**핵심 기능:**
- [ ] 회원 관리 대시보드
- [ ] 운동 프로그램 배포
- [ ] 실시간 피드백
- [ ] 결제 연동 (월 $20~)
