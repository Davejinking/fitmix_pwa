# Iron Log - 게이미피케이션 운동 기록 앱

듀오링고 스타일의 게이미피케이션을 적용한 Flutter 운동 기록 앱입니다.

## 🎯 주요 기능

- **운동 기록**: 세트, 무게, 횟수 추적
- **템포 모드**: TTS/비프음/무음 + 햅틱 피드백
- **게이미피케이션**: XP, 레벨, 리그, 파워, 업적
- **스트릭**: 연속 운동 일수 추적
- **다국어 지원**: 한국어, 영어, 일본어
