## 🎨 색상 및 스타일 가이드

### 선택되지 않은 항목
```
┌─────────────────────────────────────┐
│ [☐] 🏋️ Exercise Name               │
│     Body Part • Equipment           │
└─────────────────────────────────────┘

체크박스: 투명 배경 + 회색 테두리 (Colors.white24)
텍스트: 흰색 (IronTheme.textHigh)
테두리: 회색 (IronTheme.surface)
```

### 선택된 항목
```
┌─────────────────────────────────────┐
│ [✓] 🏋️ Exercise Name               │ ← 파란색
│     Body Part • Equipment           │
└─────────────────────────────────────┘

체크박스: 파란색 배경 (IronTheme.primary) + 흰색 체크
텍스트: 파란색 (IronTheme.primary)
테두리: 파란색 (IronTheme.primary, 2px)
```

### 선택 헤더
```
┌─────────────────────────────────────┐
│ 3 SELECTED        [☑] DESELECT ALL  │
└─────────────────────────────────────┘

배경: 검정 (Colors.black)
텍스트: 흰색, Courier 폰트, 굵게
테두리: 하단 회색 라인 (Colors.white24)
```

### 하단 확인 버튼
```
┌─────────────────────────────────────┐
│     ✓ 3 EXERCISES SELECTED          │ ← 회색 텍스트
│  ┌───────────────────────────────┐  │
│  │  + ADD 3 EXERCISES            │  │ ← 파란색 버튼
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘

요약: 회색 텍스트 (Colors.white70)
버튼: 파란색 배경 (IronTheme.primary) + 흰색 텍스트
```

## 🔄 인터랙션 플로우

### 플로우 1: 단일 선택
```
1. 초기 상태
   [☐] Bench Press
   
2. 탭 (HapticFeedback.lightImpact)
   [✓] Bench Press  ← 파란색으로 변경
   
3. 하단 버튼 표시
   [+ ADD 1 EXERCISE]
   
4. 버튼 탭 (HapticFeedback.mediumImpact)
   → 콜백 호출
   → 화면 닫힘
```

### 플로우 2: 다중 선택
```
1. 초기 상태
   [☐] Bench Press
   [☐] Squat
   [☐] Deadlift
   
2. Bench Press 탭
   [✓] Bench Press  ← 선택
   [☐] Squat
   [☐] Deadlift
   헤더: "1 SELECTED"
   
3. Squat 탭
   [✓] Bench Press
   [✓] Squat        ← 선택
   [☐] Deadlift
   헤더: "2 SELECTED"
   
4. Deadlift 탭
   [✓] Bench Press
   [✓] Squat
   [✓] Deadlift     ← 선택
   헤더: "3 SELECTED"
   
5. 버튼 탭
   [+ ADD 3 EXERCISES]
   → 3개 운동 추가됨
```

### 플로우 3: 전체 선택
```
1. [SELECT ALL] 버튼 탭
   → 모든 항목 선택됨
   → 버튼 텍스트: "DESELECT ALL"
   
2. 원하지 않는 항목 탭하여 해제
   [✓] Bench Press
   [☐] Incline Press  ← 해제
   [✓] Decline Press
   
3. [ADD N EXERCISES] 탭
   → 선택된 항목만 추가
```

### 플로우 4: 필터 + 선택
```
1. [CHEST] 필터 선택
   → 가슴 운동만 표시
   
2. [BARBELL] 장비 필터 선택
   → 바벨 가슴 운동만 표시
   
3. [SELECT ALL] 탭
   → 필터된 모든 운동 선택
   
4. [ADD N EXERCISES] 탭
   → 바벨 가슴 운동 전체 추가
```

## 💡 UX 개선 포인트

### 1. 명확한 시각적 구분
- ✅ 선택된 항목: 파란색 + 체크 표시
- ✅ 미선택 항목: 회색 + 빈 체크박스
- ✅ 테두리 색상으로 상태 구분

### 2. 실시간 피드백
- ✅ 선택 시 즉시 햅틱 피드백
- ✅ 헤더 카운터 실시간 업데이트
- ✅ 하단 버튼 텍스트 동적 변경

### 3. 효율적인 선택
- ✅ 전체 선택/해제 버튼
- ✅ 필터와 연동된 선택
- ✅ 검색 결과에서 바로 선택

### 4. 명확한 액션
- ✅ 큰 터치 영역 (전체 행)
- ✅ 명확한 버튼 레이블
- ✅ 선택 개수 표시
