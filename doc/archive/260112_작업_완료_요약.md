## 🎯 핵심 변경사항

### Hybrid Noir 전략이란?
디자인 요소는 **영어로 고정**, 사용성 요소는 **다국어 지원**하는 새로운 접근법입니다.

### 영어 고정 (Design Elements)
```
WEEKLY STATUS
MONTHLY GOAL  
EXERCISES
SESSION READY
SESSION COMPLETE
NO ACTIVE SESSION
STATUS: RESTING
```

### 다국어 지원 (Usability Elements)
```
버튼: 운동 시작 / Start Workout / ワークアウト開始
필터: 밀기 / Push / プッシュ
검색: 루틴 검색 / Search Routine / ルーティン検索
```
