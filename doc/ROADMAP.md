# 🏋️ Iron Log (アイアンログ) 프로젝트 로드맵

**프로젝트명:** Iron Log (아이언 로그 / アイアンログ)  
**타겟 시장:** 글로벌 (영어, 일본어)  
**최종 업데이트:** 2026-01-04

---

## 📋 프로젝트 개요

**Iron Log**는 개인 운동 기록에 특화된 심플하고 강력한 웨이트 트레이닝 앱입니다.
- **Iron (쇠)**: 무게, 강철 같은 의지
- **Log (기록)**: 운동 일지, 데이터 로그

---

## 🎯 개발 단계 (Development Phases)

### 📍 Phase 1: Athlete 모드 & MVP (현재 단계) ⭐
> **목표:** 혼자 운동하는 사용자를 위한 완벽한 기록 앱

**핵심 기능:**
- [x] 운동 계획 및 기록
- [x] 휴식 타이머 (애플 스타일)
- [x] 템포 모드
- [x] 게이미피케이션 (XP, 레벨, 스트릭)
- [ ] Iron Pro 구독 기능
- [ ] 고급 통계 차트
- [ ] 데이터 백업/복원

### 📍 Phase 2: Squad 모드 (확장)
> **목표:** 커뮤니티 및 경쟁 기능

**핵심 기능:**
- [ ] 친구 추가 및 팔로우
- [ ] 운동 기록 공유
- [ ] 주간/월간 리더보드
- [ ] 챌린지 참여

### 📍 Phase 3: Coach 모드 (수익화 확장)
> **목표:** 트레이너를 위한 SaaS 플랫폼

**핵심 기능:**
- [ ] 회원 관리 대시보드
- [ ] 운동 프로그램 배포
- [ ] 실시간 피드백
- [ ] 결제 연동 (월 $20~)

---

## 💰 수익화 전략 (Monetization)

### Free (무료)
- 운동 기록 무제한
- 기본 통계
- 휴식 타이머
- 템포 모드

### Iron Pro (구독) - $3.99/월 (400엔)
- 고급 차트 및 분석
- 테마 변경
- 클라우드 백업
- 광고 제거
- 우선 지원

### Coach Pro (SaaS) - $20~/월 (Phase 3)
- 회원 관리 (최대 50명)
- 프로그램 배포
- 수익 분석
