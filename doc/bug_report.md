- **실제:** 홈 화면 일부만 번역되고, 나머지 페이지는 한국어로 표시됨

- **상태:** 🆕 신규
- **상세:**
  - 파워 상점: 전체 하드코딩 (파워 상점, 아이템, 스트릭 프리즈 등)
  - 업적 페이지: 전체 하드코딩 ('업적', '현재 스트릭', '총 운동', '총 볼륨', '달성 완료!', '미달성' 등)
  - 분석 페이지: 전체 하드코딩 (신체 밸런스 분석, 분석 결과 등)
  - 캘린더 페이지: 대부분 하드코딩 (운동 계획, 세트 추가 등)
  - 라이브러리 페이지: 부위명 하드코딩 (가슴, 등, 하체 등)
  - 리그 이름: 하드코딩 (실버, 골드 등) - gamification.dart

### BUG-005: 템포 모드 시작 전 준비 시간 없음 (UX 문제)
- **화면:** 운동 계획 → 템포 모드
- **재현:** 템포 모드 버튼 탭
- **예상:** 준비 카운트다운 후 템포 시작 (자세 세팅 시간 확보)
- **실제:** 버튼 누르자마자 바로 "DOWN" 시작
- **심각도:** 🟡 Medium
- **상태:** 🆕 신규
- **문제점:**
  - 핸드폰 주머니에 넣을 시간 없음
  - 바벨 잡고 자세 세팅할 시간 없음 (보통 10~15초 소요)
  - 이어폰/햅틱에만 의지해야 하는데 준비 시간이 없음
- **개선안:**
  - 준비 카운트다운 추가 (10초/15초/20초 선택)
  - 시작 신호음 (삐-삐-삐-삐빅!)
  - 준비 중 진동 피드백

### BUG-006: 템포 시간 설정 시 직접 입력 불가 (UX 문제)
- **화면:** 운동 계획 → 템포 설정 모달
- **재현:** 내리는 시간/올리는 시간을 1초에서 10초로 변경 시도
- **예상:** 숫자 탭하면 직접 입력 가능
- **실제:** +/- 버튼으로만 조절 가능 (9번 클릭 필요)
- **심각도:** 🟢 Low
- **상태:** 🆕 신규
- **개선안:**
  - 숫자 영역 탭 시 숫자 키패드로 직접 입력
  - +/- 버튼은 유지 (1~2초 미세 조정용)
  - 프리셋 버튼 추가 고려 (2초/4초/6초/8초)

### BUG-007: 템포 모드 팝업 표시 옵션 없음 (UX 문제)
- **화면:** 운동 계획 → 템포 모드 실행 중
- **재현:** 템포 모드 시작 후 핸드폰을 주머니에 넣고 운동
- **예상:** 화면 안 볼 때는 팝업 불필요
- **실제:** 항상 전체 화면 팝업 표시
- **심각도:** 🟢 Low
- **상태:** 🆕 신규
- **사용 시나리오:**
  - 화면 보면서 운동 (덤벨 컬 등) → 팝업 유용
  - 화면 안 보고 운동 (벤치프레스, 스쿼트 등) → 팝업 의미 없음
- **개선안:**
  - 팝업 표시 옵션: 전체화면 / 하단 미니바 / 숨김
  - "무음 모드" 선택 시 자동으로 팝업 숨김 (햅틱만 사용)

### BUG-008: "나의 목표" 수정 시 프로필 정보 화면으로 이동 (설계 오류)
- **화면:** 홈 → 나의 목표 카드 → 수정 버튼
- **재현:** "나의 목표"에서 수정 버튼 탭
- **예상:** 운동 목표 설정 화면 (목표 유형, 기간, 수치 등)
- **실제:** 프로필 정보 입력 화면 (체중, 키, 생년월일, 성별)
- **심각도:** 🟡 Medium
- **상태:** 🆕 신규
- **문제점:**
  - "목표"와 "프로필 정보"는 다른 개념
  - 현재 목표는 단순히 운동 일수/볼륨만 표시
- **개선안:**
  - 목표 설정 화면 신규 개발:
    - 목표 유형: 근육 증가 / 체중 감량 / 체력 향상 / 3대 중량 증가
    - 기간: 1개월 / 3개월 / 6개월 / 1년
    - 구체적 수치: 벤치 100kg, 체중 -5kg, 월 볼륨 10만kg 등
  - 프로필 정보는 설정 > 프로필에서 수정
  - 목표 달성률 추적 및 시각화
